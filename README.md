# AI Learning Lab

This repository is my personal workspace for learning AI/ML concepts and experimenting with implementations across multiple languages (starting with **Python** and **Java**).

## Goals
- Build small focused projects as I learn new topics
- Keep each project reproducible (clear setup + run instructions)
- Track notes, references, and learnings in one place

## Repository layout
- `projects/` — hands-on mini-projects (organized by language)
  - `projects/python/` — Python projects
  - `projects/java/` — Java projects
- `notes/` — learning notes, links, summaries
- `scripts/` — helper scripts (optional)
- `datasets/` — local datasets (optional; may be excluded from git)

## How projects are organized
Each project lives in its own folder and should include:
- a `README.md` with what it does and how to run it
- source code in `src/`
- tests in `tests/` (where applicable)
- dependency files (e.g., `requirements.txt`, `pyproject.toml`, `pom.xml`)

## Creating a new project
1. Copy the appropriate template:
   - Python: `projects/python/000-template` → `projects/python/<nnn>-<project-name>`
   - Java: `projects/java/000-template` → `projects/java/<nnn>-<project-name>`
2. Update the new project's `README.md`
3. Implement and iterate
